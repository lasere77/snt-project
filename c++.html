le c++ c'est:

le c++ est un langage de développement orienté objet, il est réputé pour être compliqué surtout a cause des pointeurs,
les pointeurs sont des variables qui vont stocker l'adresse d'un segment de la ram par exemple: 000000E9C78FF924 qui aura pour contenus 10.
mais ce langage n'est pas que compliqué il est aussi très performant,
il vous donnera une bonne vitesse d'exécution car c'est un langage qui n'est pas haut niveaux, il est plus proche de la machine.
