java

java est un langage orienté objet qui a pour spécifitée que tout peut devenir un objet, un objet c'est une classe auquelle on va mettre des propriétés name, health, shild, damage...
et en java dans chacun des fichiers il y a une classe ce qui fait que chaques fichiers peut devenir un objet mais il faut que dans ce fichier il y soit un constructeur c'est là ou l'on va mettre les arguments.
à noter que java n'est pas forcément un langage haut niveaux.
